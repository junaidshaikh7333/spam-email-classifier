# Spam Email Classifier

This project is a Machine Learning based Spam Email Classifier built using Python.
It classifies messages as **Spam** or **Not Spam (Ham)** using text analysis.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Naive Bayes Algorithm

## Dataset
SMS Spam Collection Dataset containing labeled spam and ham messages.

## How to Run
```bash
python spam_classifier.py
